    如何解决npm安装包失败或者过慢问题
    1. Node环境安装

    • 安装地址 (opens new window)，注意：请下载长期支持版本(LTS)。

    • 安装完成后，在终端输入: node -v，查看安装的版本号。

    • 默认会安装npm，可以使用npm -v，查看npm(Node Package Manager)的版本号。

    1. 安装nrm
    • nrm用来管理npm镜像源的工具，很多时候，我们使用npm安装包的时候太慢

    windows和Mac电脑是不同安装方式：

    • Windows: npm install -g nrm

    • Mac: sudo npm install -g nrm

    • 注意： 如果出现nrm安装失败，那就多安装几次就会成功

    1. 安装nrm成功后
    • 在终端输入：nrm ls命令，可以查看我们有哪些源可以使用，winodws下默认使用第一个，Mac下*指的是默认使用的源。

    • 我们可以查看每个源的下载速度，使用nrm test 源名，源名指的是链接最前面的那个单词

    • 我们就可以使用nrm use 源名来切换比较快的源。

    • Mac下我们可以通过nrm current命令查看当前使用的源是哪个。

    1. 如果我们想要添加自己公司的源

    • 我们可以使用nrm add 源名 公司源地址，例如nrm add zss http://192.168.0.222:8080，其中zss就是源名，后面的链接就是源的链接地址。

    • 如果我们不想要这个源，可以使用nrm del zss删除我们添加的源，zss是我添加的源名。
